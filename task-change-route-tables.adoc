= 在多個AZs中變更HA配對的路由表
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以修改AWS路由表、其中包含部署在多個AWS可用性區域（AZs）中之HA配對的浮動IP位址路由。如果新的 NFS 或 CIFS 用戶端需要存取 AWS 中的 HA 配對、您可以這麼做。

.步驟
. 在工作環境中、按一下功能表圖示、然後按一下 * 資訊 * 。
. 按一下 * 路由表 * 。
. 修改所選路由表的清單、然後按一下「 * 儲存 * 」。


.結果
BlueXP會傳送AWS要求來修改路由表。
